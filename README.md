## Requirements

- Node V12+

## Install, Start & availables commands

```shell
# Install required modules
yarn

# Run app
yarn start

# Build
yarn build

# Run tests
yarn test

# Run prod optimized version
yarn start:prod

# Check code against ESLint
yarn eslint

# Lint src folder
yarn lint

# lint ant fix src folder
yarn lint:fix

# lint and fix css files
yarn lint:css

# Generate component, container
yarn generate

# Prettify src
yarn prettify
```
